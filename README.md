# TransitOps — Smart Transport Operations Platform

TransitOps is a centralized transport operations platform that digitizes the
full lifecycle of a logistics fleet — vehicle registration, driver management,
trip dispatch, maintenance, and fuel/expense tracking — replacing manual
spreadsheets and logbooks with real-time, rule-enforced workflows.

Built for the Odoo Hackathon (8-hour build) by a 4-developer team.

## Problem
Logistics companies relying on spreadsheets face scheduling conflicts,
underutilized vehicles, missed maintenance, expired driver licenses, and no
real-time visibility into operational cost.

## Solution
A single platform where dispatching a trip automatically validates vehicle
capacity, driver license status, and availability — then cascades status
changes (Available → On Trip → Available) across vehicles and drivers
automatically, with live cost and efficiency reporting.

## Roles
Fleet Manager · Driver · Safety Officer · Financial Analyst
